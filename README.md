# E-Commerce-Website
A fully responsive and interactive shopping website built using core web technologies. This project simulates a simple e-commerce experience with product listings, categories, cart management, and a clean multi-page structure
=> Features
   * Core Functionalities

   Multi-Page Design — Includes Home, Products, Cart, and Contact pages.

   Product Display Cards — Clean layout with images, descriptions, and pricing.

   Responsive Layout — Built using CSS Flexbox & Grid for all devices.

 * JavaScript Features

   Add to Cart System — Add, remove, and update item quantities.

   Real-Time Cart Updates — Total price and item count update instantly.

   localStorage Support — Cart data remains saved even after refreshing.

   DOM Manipulation — Dynamic product and UI updates without reloading pages.

 * UI / UX Enhancements

   Smooth hover effects and transitions

   Clean visual theme with consistent colors and typography

   Mobile-friendly navigation bar
 => Technologies Used

    * HTML5 — Semantic page structure

    * CSS3 — Responsive styling & animations

    * JavaScript (ES6) — Logic, interactions & data handling
